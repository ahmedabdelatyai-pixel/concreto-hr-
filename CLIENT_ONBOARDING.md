
## 📝 إضافة وظيفة جديدة

### الخطوات:

1. اذهب إلى **"الوظائف" → "إضافة وظيفة جديدة"**
2. املأ البيانات:
   - **عنوان الوظيفة (عربي وإنجليزي)**
   - **القسم/الإدارة**
3. أضف أسئلة مخصصة (اختياري):
   - اكتب السؤال
   - اختر نوعه (تقني، سلوكي، مختلط)
   - اضغط "إضافة"
4. اضغط **"حفظ الوظيفة"**

### مثال:

```
عنوان الوظيفة (AR): مهندس برمجيات
عنوان الوظيفة (EN): Software Engineer
القسم: تقنية المعلومات

أسئلة مخصصة:
- [تقني] كم سنة خبرة لديك في React؟
- [سلوكي] كيف تتعامل مع ضغط العمل؟
```

## 👥 إضافة أعضاء فريق

1. اذهب إلى **"الإعدادات → إدارة الفريق"**
2. اضغط **"➕ إضافة عضو"**
3. ملأ البيانات:
   - **الاسم الكامل**
   - **البريد الإلكتروني**
   - **الصلاحية:**
     - `HR` - إدارة الوظائف والمتقدمين
     - `Recruiter` - عرض المتقدمين فقط
     - `Viewer` - عرض القراءة فقط

## 📊 قراءة نتائج التقييم

كل متقدم يحصل على:

### **الدرجة الإجمالية** (من 100)
- **80-100:** ممتاز ✅
- **60-79:** جيد ⚠️
- **أقل من 60:** ضعيف ❌

### **المعايير الثلاثة:**
1. **السلوك (40 نقطة):** الأداء والمعرفة التقنية
2. **الموقف (30 نقطة):** الالتزام والمسؤولية
3. **الشخصية (30 نقطة):** التواصل والقيادة

### **تحليل DISC:**
- **D (Dominant):** قيادي وحاسم
- **I (Influence):** متحدث وجذاب
- **S (Steadiness):** مستقر وموثوق
- **C (Conscientiousness):** دقيق وتفاني

## 🎓 نصائح للحصول على نتائج أفضل

### 1. كن واضح في الأسئلة
❌ "ما رأيك في العمل؟"
✅ "وصف موقف واجهت فيه مشكلة تقنية طارئة وكيف حلتها؟"

### 2. ركز على الوظيفة
- أضف أسئلة تفصل بين المرشحين الجيدين والممتازين
- اسأل عن خبرة عملية محددة

### 3. عريف الطعمة
- شارك الوظيفة من مصادر متعددة
- استخدم كلمات واضحة ودقيقة

### 4. تابع المرشحين
- راجع النتائج يومياً
- تواصل مع المرشحين الممتازين بسرعة

## ❓ أسئلة شائعة

**س: كم وقت تأخذ المقابلة؟**
ج: حوالي 20-25 دقيقة (10 أسئلة × دقيقتان لكل سؤال)

**س: هل يمكن تعديل الوظيفة بعد إنشاؤها؟**
ج: نعم، لكن لا يؤثر على المتقدمين السابقين

**س: ماذا لو لم يجب المتقدم على جميع الأسئلة؟**
ج: يتم حساب نتيجتك حسب الإجابات تلقائياً

**س: هل يمكن مشاركة التقرير مع المتقدم؟**
ج: حالياً يتلقى المتقدم النتيجة الإجمالية فقط، التقرير الكامل لك فقط
